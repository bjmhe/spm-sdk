## Install

<!-- automd:pm-install dev auto=false -->

```sh
# npm
npm install -D @jd/spm-sdk

# yarn
yarn add -D @jd/spm-sdk

# pnpm
pnpm add -D @jd/spm-sdk

# bun
bun install -D @jd/spm-sdk

# deno
deno install --dev npm:@jd/spm-sdk

# vp
vp install -D @jd/spm-sdk
```
